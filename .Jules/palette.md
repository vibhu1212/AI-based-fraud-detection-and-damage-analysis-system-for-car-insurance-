## 2024-03-24 - Interactive Component Accessibility\n**Learning:** React custom interactive components (such as `div` elements acting as buttons or drop zones) must include `role="button"`, `tabIndex={0}`, an `onKeyDown` handler for 'Enter' and 'Space' keys, and `:focus-visible` CSS styles to guarantee keyboard accessibility.\n**Action:** Add full keyboard support to custom interactive elements and ensure focus is visible.
